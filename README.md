A information gathering tool for Linux &amp; Termux

## Recomended Devices
This tool is usually created for Linux & Termux

## Installation
1) `git clone https://github.com/krishtechstudio/RedSpy`
2) `cd RedSpy`
3) `chmod +x *`
4) `./install.sh`
5) `RedSpy`

## Run
To run the tool usually type `RedSpy`
